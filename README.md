# 🗣️ Voice Disorder Detection & Classification

A machine learning project focusing on the medical detection and classification of voice disorders using advanced feature extraction and a custom PyTorch OS-ELM model.

### ✨ Features
- **MFCC Feature Extraction:** Extracts Mel-frequency cepstral coefficients from raw audio files.
- **SMOTE Balancing:** Handles severe class imbalances common in medical datasets.
- **OS-ELM Model:** Implementation of Online Sequential Extreme Learning Machine for fast and accurate classification.
- **GPU Optimization:** Accelerated training pipeline using PyTorch.

### 🛠️ Tech Stack
- Python, PyTorch, Librosa
- Scikit-Learn, Imbalanced-Learn
- Jupyter Notebooks for EDA and Training
